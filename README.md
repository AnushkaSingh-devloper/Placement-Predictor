# 🎓 Placement Predictor 

This is a simple Machine Learning project built for learning purposes.  
It predicts whether a student is likely to be placed based on two features:
- CGPA  
- IQ  

📊 About the Project
- Built using a **toy dataset** for practice and understanding ML concepts  
- Model used: Logistic Regression  
- Focus: Learning model training, visualization, and deployment  

🚀 Tech Stack
- Python  
- Scikit-learn  
- NumPy  
- Streamlit  

▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt  

2. Run the app:
   streamlit run app.py  

🎯 Output
The app takes CGPA and IQ as input and predicts:
- ✅ Placed  
- ❌ Not Placed  

⚠️ Note
This project is created for **learning and demonstration purposes only**.  
The dataset is small and does not represent real-world accuracy.
